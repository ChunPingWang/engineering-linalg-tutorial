### 1.3 內積（Dot Product）

**數學概念**
- 代數定義：$\vec{a} \cdot \vec{b} = \sum a_i b_i$
- 幾何意義：$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$
- NumPy：`np.dot(a, b)`、`a @ b`

**物理例子 💡 功與能量**
- **功的計算**：$W = \vec{F} \cdot \vec{d}$（力與位移的內積）
  - 水平施力推斜坡：只有平行位移方向的分力做功
  - 磁力永不做功：因為 $\vec{F}_{磁} \perp \vec{v}$，內積為零
- **光線與法向量夾角**：太陽能板最佳擺放角度
- **信號投影**：感測器量測值投影到參考方向

**實作清單**

- [ ] 驗證：磁力在圓周運動中做功始終為零

### 1.4 外積（Cross Product）

**數學概念**
- 3D 外積定義與行列式計算法
- 右手定則
- NumPy：`np.cross(a, b)`

**物理例子 🔩 力矩與旋轉**
- **力矩（Torque）**：$\vec{\tau} = \vec{r} \times \vec{F}$
  - 扳手鎖螺絲：施力點位置向量 × 施力向量 = 扭矩
- **角動量**：$\vec{L} = \vec{r} \times \vec{p} = \vec{r} \times m\vec{v}$
- **螺旋槳 / 陀螺儀**：角速度向量方向即旋轉軸

**實作清單**

- [ ] 扳手效率模擬：角度對力矩大小的影響曲線
- [ ] 驗證：$|\vec{a} \times \vec{b}| = |\vec{a}||\vec{b}|\sin\theta$ = 平行四邊形面積

### 3.3 超定系統（Over-determined）— 最小平方解

**數學概念**
- 方程式比未知數多：無精確解

- Normal Equation：$A^T A \vec{x} = A^T \vec{b}$

**物理例子 📡 量測與校正**
- **GPS 定位**：超過 4 顆衛星的量測，最小平方估計位置
- **感測器融合**：多個加速度計量測同一物理量，取最佳估計

### 3.4 欠定系統（Under-determined）— 最小範數解

**數學概念**
- 方程式比未知數少：無窮多解
- 偽逆求最小範數解：$\vec{x} = A^+ \vec{b}$

**物理例子 🤖 冗餘機器手臂**
- 7-DOF 機器手臂執行 6-DOF 任務：多餘一個自由度
- 最小能量移動路徑：在無窮多解中選能量最小的

## Module 4 — 向量空間與自由度

> **對應程度**：大學線代基礎，搭配機械系統直觀理解

### 4.1 線性獨立與相依

**數學概念**
- 線性組合的定義
- 線性相依 = 某向量可由其他向量表示
- 判斷方法：行列式 = 0 ↔ 相依

**物理例子 🔗 約束與自由度**
- **機構分析**：三根桿件在同一平面的力若線性相依，無法提供全方向約束
- **過約束結構（Statically Indeterminate）**：約束方程式線性相依造成超靜定
- **慣性主軸**：三個主軸向量必須線性獨立（正交）

**實作清單**
- [ ] 視覺化 2D / 3D 線性相依向量（共線、共面）
- [ ] 桁架約束分析：判斷靜定 vs 超靜定
- [ ] 使用 `np.linalg.matrix_rank` 判斷相依性

---

### 4.2 矩陣的秩（Rank）

**數學概念**
- 秩 = 線性獨立的行（或列）數
- `np.linalg.matrix_rank(A)`
- 秩-零化度定理：$\text{rank}(A) + \text{nullity}(A) = n$（列數）

**物理例子 🦾 機構自由度**
- **Grübler 公式的矩陣版本**：約束矩陣的秩 = 有效約束數
- 連桿機構的自由度 = n - rank(Jacobian)
- 結構冗餘度分析

**實作清單**

- [ ] 秩不足矩陣的物理意義演示

---

### 4.3 零空間（Null Space）

**數學概念**
- $\text{Null}(A) = \{\vec{x} : A\vec{x} = \vec{0}\}$
- 計算：`scipy.linalg.null_space(A)`

**物理例子 🌀 剛體運動**
- **剛體零空間**：結構整體平移、旋轉（不產生內力的運動）
- 飛行器的「自由模態」：結構在自由空間中的無約束運動
- 化學反應網路的守恆量

**實作清單**

- [ ] 驗證秩-零化度定理

### 4.4 投影（Projection）與正交化

**數學概念**
- 向量在子空間的正交投影
- Gram-Schmidt 正交化過程
- 投影矩陣：$P = A(A^TA)^{-1}A^T$

**物理例子 📡 訊號處理**
- **震動訊號分解**：將量測訊號投影到各模態方向
- **雷達目標追蹤**：量測投影到觀測方向
- **加速度計去除重力分量**：從總加速度投影去除重力向量

### 5.2 QR 分解

**數學概念**
- $A = QR$（正交矩陣 × 上三角矩陣）
- `np.linalg.qr(A)`
- 與最小平方的關係

**物理例子 📐 姿態估算**
- **旋轉矩陣正交化**：量測到的姿態矩陣因噪聲不完全正交，用 QR 修正
- **惯性導航（INS）**：定期對姿態矩陣做 QR 正交化防止漂移
- 最小平方法的數值穩定實作

**實作清單**
- [ ] 量測噪聲的旋轉矩陣：使用 QR 正交化後驗證 $Q^TQ = I$
- [ ] 以 QR 分解解最小平方（比 Normal Equation 更穩定）
- [ ] 驗證：Gram-Schmidt 等價於 QR 分解

---

### 5.3 Cholesky 分解

**數學概念**
- $A = LL^T$（對稱正定矩陣專用）
- `np.linalg.cholesky(A)`
- 效率是 LU 的兩倍

**物理例子 🌡️ 熱傳導穩態**
- **穩態熱傳導方程**：$K\vec{T} = \vec{q}$，$K$ 是對稱正定導熱矩陣
- **Kalman Filter 共變異數矩陣更新**（預告現代控制）
- 有限元素質量矩陣 $M$ 的分解

**實作清單**
- [ ] 一維穩態熱傳導問題：左端 200°C，右端 20°C，求中間節點溫度
- [ ] Cholesky vs LU 求解時間比較
- [ ] 驗證正定矩陣的 Cholesky 存在性

---

### 5.4 SVD — 奇異值分解（核心）

**數學概念**
- $A = U \Sigma V^T$
- 奇異值 = 資料的「重要性排序」
- `np.linalg.svd(A)`

**物理例子 🖼️ 多重應用**

**① 圖像壓縮（直覺入門）**
- 灰階影像矩陣的 SVD
- 保留前 $k$ 個奇異值重建影像
- 壓縮率 vs 影像品質曲線

**② 結構模態分析**
- 實驗量測的頻率響應矩陣（FRF Matrix）
- SVD 提取主要振動模態（模態截斷）
- 識別結構的固有頻率與模態形狀

**③ 流場 POD（Proper Orthogonal Decomposition）**
- 流速場數據矩陣的 SVD
- 主要流動模式提取（渦流、剪切層等）

**④ 機器手臂可操作性橢球**
- 雅可比矩陣 $J$ 的 SVD
- 奇異值 = 各方向的操作靈活度
- 奇異位形 = 最小奇異值趨近零

**實作清單**
- [ ] 灰階圖像 SVD 壓縮：滑桿控制保留奇異值數量
- [ ] FRF 矩陣模態識別（模擬 3 自由度振動系統）
- [ ] 2D 流場快照矩陣 SVD：提取主要流動模式
- [ ] 機器手臂 Jacobian SVD：可操作性橢球視覺化
- [ ] 驗證：$A = U \Sigma V^T$ 重組誤差 < 1e-12

## Module 6 — 特徵值與動態系統

> **對應程度**：大學線代 + 微分方程基礎

### 6.1 特徵值與特徵向量定義

**數學概念**
- $A\vec{v} = \lambda\vec{v}$
- 特徵多項式 $\det(A - \lambda I) = 0$
- `np.linalg.eig(A)`（一般）、`np.linalg.eigh(A)`（對稱）

**物理例子 🎵 自然振動**
- 弦樂器振動：不同模態 = 不同特徵向量
- 特徵值 = 頻率平方，特徵向量 = 振型
- 大樓地震響應分析

**實作清單**
- [ ] 2-DOF 彈簧-質量系統：建立剛度矩陣，求固有頻率與振型
- [ ] 振型動畫：各模態的同時運動視覺化
- [ ] 驗證：$A \vec{v}_i = \lambda_i \vec{v}_i$（各特徵對）

---

### 6.2 對稱矩陣的特徵分解

**數學概念**
- 對稱矩陣：實數特徵值、正交特徵向量
- 譜定理：$A = Q \Lambda Q^T$
- 正定矩陣：所有特徵值 > 0

**物理例子 🔧 慣性張量**
- **剛體慣性張量（Inertia Tensor）**：
  $$I = \begin{bmatrix} I_{xx} & -I_{xy} & -I_{xz} \\ -I_{yx} & I_{yy} & -I_{yz} \\ -I_{zx} & -I_{zy} & I_{zz} \end{bmatrix}$$
  - 必為對稱矩陣
  - 特徵值 = 主慣性矩
  - 特徵向量 = 主慣性軸（Principal Axes）
- 飛行器或衛星繞主軸旋轉最穩定
- 主應力 / 主應變分析（莫爾圓的矩陣版本）

**實作清單**
- [ ] 任意 3D 剛體慣性張量計算與主軸求解
- [ ] 視覺化慣性橢球（Inertia Ellipsoid）
- [ ] 主應力張量分解：從 3D 應力狀態求主應力方向
- [ ] 驗證：$Q^T A Q = \Lambda$（對角化）

---

### 6.3 矩陣的穩定性分析

**數學概念**
- 動態系統 $\dot{\vec{x}} = A\vec{x}$
- 穩定條件：所有特徵值實部 < 0
- 解：$\vec{x}(t) = e^{At}\vec{x}_0$

**物理例子 🚁 控制系統**
- **倒擺穩定性**：特徵值有正實部 → 不穩定 → 需要控制器
- **飛行器縱向動態**：短週期模態、長週期模態的特徵值
- **電路 RC 衰減**：特徵值為 $-1/RC$（負實數，穩定）

**實作清單**
- [ ] 倒擺線性化模型：計算特徵值，確認不穩定
- [ ] 加入 PD 控制後：特徵值移至左半平面（穩定化）
- [ ] 相平面圖（Phase Portrait）：穩定焦點 vs 鞍點 vs 極限環
- [ ] `scipy.linalg.expm`：矩陣指數 $e^{At}$ 數值計算

---

### 6.4 Markov 鏈與穩態

**數學概念**
- 轉移矩陣（Stochastic Matrix）
- 穩態：最大特徵值 = 1 對應的特徵向量

**物理例子 🌊 熱平衡與擴散**
- **熱擴散 Markov 模型**：格點間的熱量轉移矩陣
- **分子擴散**：粒子在格點間隨機跳躍的穩態分佈
- 網路流量路由的穩態

**實作清單**
- [ ] 5 格點一維熱擴散 Markov 模型
- [ ] 長時間演化後收斂到穩態
- [ ] 驗證：穩態 = 對應特徵值 1 的歸一化特徵向量

## Module 7 — 最小平方與資料擬合

### 7.1 線性迴歸的矩陣解

**數學概念**

### 7.2 正規化（Regularization）

**數學概念**
- Ridge Regression（L2）：$\min ||A\vec{x} - \vec{b}||^2 + \lambda||\vec{x}||^2$
- 解：$(A^TA + \lambda I)^{-1}A^T\vec{b}$

**物理例子 🌡️ 熱傳逆問題**
- **逆向熱傳導問題**：已知表面溫度，估計內部熱源
- 病態問題（Ill-posed）：正規化防止解發散
- 量測噪聲放大問題

**實作清單**
- [ ] 逆熱傳問題：條件數極大，直接求解發散
- [ ] 加入 Ridge 正規化：穩定解的求取
